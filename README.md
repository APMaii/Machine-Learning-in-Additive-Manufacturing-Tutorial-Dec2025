This repository contains all materials for the **Machine Learning in Additive Manufacturing (AM)** tutorial held in **December 2025** at the **AMpro Research Group, Politecnico di Torino**.

The tutorial introduces how modern machine-learning techniques can be applied to **Laser Powder Bed Fusion (LPBF)**, **Electron Beam Melting (EBM)**, and other AM technologies.

---

## ðŸŽ¯ Objectives

Participants will learn how to:

- Understand the structure of Additive Manufacturing datasets  
- Preprocess AM data (sensor data, images, process parameters, microstructure, defects, etc.)  
- Apply **supervised** and **unsupervised** machine learning methods  
- Build ML models for:
  - Melt pool behavior  
  - Porosity and defect prediction  
  - Mechanical property estimation  
  - Process parameter optimization  
- Use **deep learning** for AM images and signals  
- Interpret and evaluate machine-learning models
